# Các bước cài đặt aapanel trên Centos7
- Bước 1 : install wget 
```
yum install wget 
```
- Bước 2 : cài đặt phiên bản aapanel
  - Tải file cài đặt 
```
wget -O install.sh http://www.aapanel.com/script/install-ubuntu_6.0_en.sh
```
  - Thực hiện cài đặt : 
  ```
  bash insatll.sh
  ```
- Đợi quá trình cài đặt hoàn tất
